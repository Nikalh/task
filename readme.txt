

Это веб-приложение создано для управления иерархической структурой сети по продаже электроники. Оно содержит API интерфейс и админ-панель для управления базой данных.

База данных создана с использованием миграций Django. В базе данных предусмотрены следующие уровни иерархической структуры:
- Завод;
- Розничная сеть;
- Индивидуальный предприниматель.

Каждое звено сети содержит следующие элементы:
- Название;
- Контакты (Email, страна, город, улица, номер дома);
- Продукты (название, модель, дата выхода продукта на рынок, поставщик, задолженность перед поставщиком в денежном выражении с точностью до копеек);
- Время создания (заполняется автоматически при создании).

Админ-панель позволяет выводить созданные объекты сети. На странице объекта сети доступны следующие возможности:
- Ссылка на "Поставщика";
- Фильтр по названию города;
- "Admin action", очищающий задолженность перед поставщиком у выбранных объектов.

Используя DRF, создан набор представлений CRUD для модели поставщика. При этом обновление поля "Задолженность перед поставщиком" через API запрещено. Добавлена возможность фильтрации объектов по определенной стране.

Права доступа к API настроены таким образом, чтобы только активные сотрудники имели доступ к API.
